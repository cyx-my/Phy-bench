Project Overview

Phy-Bench evaluates two tiers of world model capabilities:

Tier 1: Prediction

Question: Can the model apply a rule?

Task: Given first N frames (± rule description), predict next M frames

Metrics: Trajectory MSE + Physics consistency checks

Tier 2: Induction

Question: Can the model discover a rule?

Task: Given K example videos, produce a natural language rule description

Metrics: Fact coverage, bonus coverage, confusion penalty, generalization accuracy

Data Format Example

Each episode generates a states.jsonl file with per-frame physics state:

{
  "frame": 42,
  "timestamp": 1.4,
  "objects": [
    {
      "id": 0,
      "position_x": 320.5,
      "position_y": 240.1,
      "velocity_x": -50.2,
      "velocity_y": 120.0,
      "radius": 12,
      "mass": 1.0
    }
  ],
  "world_state": {
    "gravity_x": 0,
    "gravity_y": -500
  },
  "events": []
}

Quick Start Guide

1. Installation
pip install -r requirements.txt
2. Generate Example Dataset
python -m benchmark.dataset_builder \
    --worlds black_hole gravity_merry_go_round \
    --episodes-per-world 3 \
    --data-root examples/
3. Evaluate Dataset
python evaluate_minimal.py \
    --data-root examples/ \
    --worlds all
4. Test Single Episode
python evaluate_minimal.py \
    --episode examples/worlds/black_hole/episodes/ep_001/states.jsonl \
    --world-id black_hole
